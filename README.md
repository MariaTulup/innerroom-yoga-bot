# Inner Room Yoga Bot

Telegram-бот для приёма заявок и скриншотов оплаты.

## Возможности

- отвечает на `/start` приветствием и ссылкой на оплату;
- уведомляет администратора о новом пользователе;
- пересылает администратору сообщения и изображения;
- позволяет ответить клиенту командой `/reply ID текст`.

## Запуск

Нужны переменные окружения:

- `BOT_TOKEN` — токен из BotFather;
- `ADMIN_ID` — числовой Telegram ID администратора;
- `DATABASE_PATH` — необязательно, по умолчанию `data/bot.sqlite3`.

Запуск:

```bash
BOT_TOKEN="..." ADMIN_ID="..." python bot.py
```
